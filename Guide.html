1. About This Tool

Most board handbooks are written once, printed twice, and never opened again. This tool flips the workflow: instead of a static document, you maintain a living set of structured inputs (org details, board roster, policies, procedures, programs, financials, committees, document inventory) — then generate a polished handbook on demand whenever you need to onboard a new member or refresh the board.

Don't write — collect

This tool isn't a word processor. It's a structured collector. You fill in each section over time; the Generator turns the inputs into a professional handbook.

2. Getting Started

Demo: demo@example.org / demo — populated with a small, well-formed sample organization.

Suggested order

  1. About — basic org info, mission, vision
  2. People — board roster, staff list, key roles
  3. Policies — start with COI, whistleblower, document retention
  4. Procedures — the operational SOPs
  5. Committees — link to or summarize your committee structure
  6. Programs & Donors — what you do, who funds it
  7. Financials — last audited financials, budget
  8. Doc Inventory — what's filed where

3. Data Entry Pages

Each section has a structured form. Fields are designed to match what board members actually need to know — not legal jargon. Tooltips and examples guide you through the trickier ones (insurance coverage, conflict of interest definitions, etc.).

You don't have to fill out everything to use the tool. Sections marked "Optional" can be skipped. The handbook generator gracefully handles missing data — sections with no content are omitted.

4. Self-Assessment

A scored questionnaire that surfaces governance gaps. Questions are organized by domain (Mission/Strategy, Fundraising, Financial Management, Board Structure, Legal/Compliance, Programs, Public Image).

Each question scores 0-3. Below-threshold answers automatically generate Action Items on the next page. Run the assessment annually — it doubles as a board self-evaluation tool.

↑ Back to top

5. Action Items

Tasks generated from the self-assessment, sorted by priority. Each captures the source question, a recommended fix, assigned owner, target date, and status. Use it as the standing agenda for governance committee meetings.

↑ Back to top

6. Build Handbook

Generates a polished handbook in two formats:

  • HTML preview — opens in a new tab, print-ready
  • Word .docx download — board-ready, editable

The generator pulls every populated section, adds a cover page, table of contents, and appendices (policies, current roster, financial highlights). Re-run whenever you make material changes.

7. Team & Pro

Multi-user editing on Pro (Free is capped at 3 team members). Activity log shows who changed what. Useful when the board secretary, ED, and chair are all maintaining the handbook together.
